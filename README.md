# Set Up React App

- npx create-react-app
  - Tailwind
    - npm install -D tailwindcss
    - npx tailwindcss init
  - Router Dom
    - npm i -D react-router-dom

# Steps

- created react app
- configure tailwindcss and router dom
- Header
- Login
